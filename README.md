## Este es el tipo de mi proyecto:
> Un detector de estafas que analiza textos o mensajes para identificar patrones sospechosos utilizando Python, procesamiento de lenguaje natural y una base de datos de ejemplos reales.

## Las bibliotecas que voy a utilizar:
- **scikit-learn** — Para entrenar modelos de clasificación y vectorización del texto.
- **pandas** — Para manejar datasets de mensajes.
- **sqlite3** — Para almacenar ejemplos de estafas y consultas.
- **re (regex)** — Para detectar patrones textuales comunes en estafas.
- **Flask (opcional)** — Si más adelante decido crear una API o interfaz web.
- **NLTK o spaCy (opcional)** — Para análisis de lenguaje natural más avanzado.
